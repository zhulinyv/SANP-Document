## Pixiv

关于上传 Pixiv, 需要将选择上传的图片或文件夹放到同一个文件夹, 例如 `./output/pixiv`

``` css
例如:
.\output
  └---\pixiv
        └---7589641258_GenshinImpact_可莉.png
        └---6594641258dwuibuib_None_None.png
        └---拉菲.png
        └---\Nahida
              └---5264942125_GenshinImpact_纳西妲.png
              └---纳西妲.png
```
